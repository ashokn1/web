Main Features

  • Responsive Design

    Cube Portfolio adapts his layout to the viewing device by using a fluid grid. This way you offer a consistent user experiences across all devices. Also you can set the number of columns based on the browser width.

  • Fully Customizable

    You can customize everytingh from grid layout, lightbox style to every possible animation. With just a little HTML, CSS ans JS you can customize the plugin to fulfill your needs.

  • 19 Starter Templates

    Cube Portfolio includes 19 starter templates. Combine features from different templates to create your perfect one. Choose what is best for you.

  • Captions

    Cube Portfolio has predefined captions, but it is very simple to add your own HMTL formatted containers inside the entries with your own CSS. You can create captions from almost any static HTML elements.

  • CSS3 Animations

    The plugin use the best browser features when available (CSS3 transition and transform, GPU acceleration) but also have nice fallbacks for legacy browsers.

  • Support & Updates For Free

    Connect to our support stuff to get your answers and stay up to date with free updates for life.

  • Permalinks (deeplinking)

    You may link to each project from address bar by adding #cbp hash to the end of URL. Optionally you can make URL automatically change when singlePage is changes.

  • Video support

    YouTube, Vimeo, Ted or self-hosted videos can be added to any item. Also, you can add video in lightbox or singlePage popup.

  • Cross-browser support

    Cube Portfolio works and has been tested on IE 8+, Chrome 10+, Mozilla 4+, iOS and Android 2.0+.

  • Grid & Slider Layout

    There are two types of layout to choose: slider or grid. Slider layout will arrage the items horizontally with navigation arrows. Grid layout place the items in optimal position based on the available vertical space.

  • Documentation

    Extensive step-by-step guide to help you with usage and customization.

  • Filtering

    Filter your portfolios, galleries, team members, blog posts or any other ordered content.
